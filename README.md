## Installation

```bash
npm install alpha-ui
```

## Usage

```tsx
import { Button } from 'alpha-ui';

function App() {
  return <Button variant="primary">Click me</Button>;
}
```

## Development

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

3. Start Storybook:

```bash
npm run storybook
```

4. Start development server:

```bash
npm run dev
```
